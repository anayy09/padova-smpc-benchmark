# Result Data Files

All files here are generated by the analysis scripts. Raw per-subject simulation outputs
(pickle files, ~15 MB total) are not included — run `run_experiments.py` to regenerate them,
then `analyze_all_results.py` to produce these tables.

---

## Primary results

**`primary_endpoint_summary.csv`**
Mean ± SD for each glycemic metric (TIR, TBR, TAR, Time<54, LBGI, HBGI, CV) broken down by
controller (SMPC, DMPC, PID), cohort (adult, adolescent, child), and scenario (standard,
variable, nocturnal). This is the source table for Table 7 in the manuscript.

**`statistical_comparison_summary.csv`**
Wilcoxon signed-rank test results for all 63 SMPC-vs-DMPC paired comparisons (7 metrics ×
3 cohorts × 3 scenarios). Columns: cohort, scenario, metric, W-statistic, p-value (raw),
q-value (Benjamini-Hochberg adjusted), rank-biserial effect size r, mean difference (SMPC −
DMPC). This is the source for Table 6 in the manuscript.

**`paper_metrics_summary.json`**
Comprehensive JSON summary of all results: per-subject metrics, cohort summaries, statistical
tests, and the figures data. Used by `build_manuscript_assets.py` to generate figures.

---

## Per-scenario comparison tables

**`metrics_comparison_{scenario}_{cohort}.csv`** (9 files)
Summary metrics for SMPC vs DMPC for each of the 9 scenario × cohort combinations. Scenarios:
standard, variable, nocturnal. Cohorts: adult, adolescent, child.

---

## Ablation study

**`targeted_ablation_child_disturbance.csv`**
Per-subject metrics for all four ablation variants (full SMPC, no-variance-penalty,
no-chance-tightening, short-horizon) run on the child cohort under variable and nocturnal
scenarios. Each row is one subject × variant combination. Source for Table 9 in the manuscript.

**`targeted_ablation_child_disturbance_summary.csv`**
Aggregated (mean ± SD) version of the ablation table. Directly maps to Table 9 numbers.

---

## Focused nocturnal child analysis

**`nocturnal_child_wilcoxon.csv`**
Wilcoxon signed-rank test results for the nocturnal child cohort specifically (TBR and LBGI
primary endpoints). Reports rank-biserial effect sizes and BH-adjusted q-values referenced
in §6.3 of the manuscript.

---

## Runtime benchmarks

**`runtime_profile_by_controller.csv`**
Per-controller average QP solve time (ms) across multiple runs and scenarios. Columns:
controller, scenario, mean_ms, std_ms, max_ms.

**`runtime_profile_summary.csv`**
Aggregated runtime statistics including EKF update time, total step time, and comparison
of SMPC vs DMPC overhead. Source for the runtime figures in §6.5 of the manuscript.
